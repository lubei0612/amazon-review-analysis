## 📦 打包步骤

### 1. 最终清理

```powershell
# 删除依赖（让客户自动安装）
Remove-Item -Recurse -Force node_modules, web\node_modules -ErrorAction SilentlyContinue

# 删除.env（用户自己配置）
Remove-Item .env -ErrorAction SilentlyContinue

# 删除内部文档（打包前）
Remove-Item "打包说明.md", "项目整理完成-交付清单.md" -ErrorAction SilentlyContinue
```

### 2. 创建压缩包

**方法1: 手动压缩**
1. 选中`maijiaplug`文件夹
2. 右键 → "发送到" → "压缩(zipped)文件夹"
3. 重命名为：`Amazon评论分析系统-客户演示包-v1.0.zip`

**方法2: PowerShell**
```powershell
Compress-Archive -Path "maijiaplug" -DestinationPath "Amazon评论分析系统-客户演示包-v1.0.zip" -Force
```

### 3. 验证压缩包

1. 解压到新目录
2. 检查文件完整性
3. 创建`.env`文件并配置
4. 运行`客户演示-一键启动.bat`
5. 确保能正常启动和使用
