# 🔧 环境配置指南

> **重要提示**：必须先配置 `.env` 文件，否则系统无法正常运行！

---

## 📝 快速配置步骤

### 步骤1：创建 `.env` 文件

在项目根目录（`D:\Users\鱼鳍\Desktop\maijiaplug`）创建 `.env` 文件：

**Windows PowerShell**:
```powershell
New-Item -Path .env -ItemType File
notepad .env
```

**Windows CMD**:
```cmd
type nul > .env
notepad .env
```

---

### 步骤2：复制以下内容到 `.env` 文件

```bash
# ========================
# Amazon评论分析系统 - 环境配置
# ========================

# ========================
# Groq AI配置 (必填)
# ========================
GROQ_API_KEY=your_groq_api_key_here
GROQ_MODEL=llama-3.1-8b-instant
GROQ_TEMPERATURE=0.3
GROQ_MAX_TOKENS=8000

# ========================
# 服务器配置
# ========================
PORT=3000
NODE_ENV=development

# ========================
# RapidAPI配置（推荐 - 速度提升10-15倍！）
# ========================
RAPIDAPI_KEY=b4c358933cmshbb9a14ce55281c4p187abdjsn32d85e42ad2b
RAPIDAPI_HOST=real-time-amazon-data.p.rapidapi.com
```

---

### 步骤3：填写 Groq API Key

1. **获取 Groq API Key**:
   - 访问：https://console.groq.com/
   - 登录/注册账号
   - 进入 "API Keys" 页面
   - 点击 "Create API Key"
   - 复制生成的 API Key

2. **填写到 `.env` 文件**:
   ```bash
   GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxx
   ```

## ❓ 常见问题

### Q1: 找不到 `.env` 文件怎么办？
A: 在项目根目录手动创建，注意文件名前面有个点 `.`

### Q2: Groq API Key 在哪获取？
A: https://console.groq.com/ → API Keys → Create API Key

### Q3: 可以用其他模型吗？
A: 可以，修改 `.env` 中的 `GROQ_MODEL` 为上表中的其他模型名称

### Q4: RapidAPI 是必须的吗？
A: 不是，RapidAPI 是可选的，用于加速爬取（推荐）
