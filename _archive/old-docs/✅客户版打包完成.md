## 🚀 客户使用流程

### 首次使用（5分钟）

1. **安装依赖**
   ```bash
   npm install
   cd web && npm install
   ```

2. **配置API**
   - 复制 `.env.example` 为 `.env`
   - 填入 `GEMINI_API_KEY` 和 `OUTSCRAPER_API_KEY`

3. **启动系统**
   - 双击 `一键启动-客户版.bat`

4. **开始使用**
   - 访问 http://localhost:3002
   - 输入ASIN并分析

### 日常使用（1分钟）

1. 双击 `一键启动-客户版.bat`
2. 等待浏览器自动打开
3. 开始分析

### 停止系统

1. 双击 `停止所有服务.bat`
2. 或关闭两个终端窗口
