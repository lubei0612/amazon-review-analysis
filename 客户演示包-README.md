## 🚀 快速开始（3步）

### 第一步：配置API密钥

1. 找到文件：`env.example`
2. 复制并重命名为：`.env`
3. 打开`.env`，填写您的API密钥：

```env
# Gemini API Key (必填)
# 获取地址：https://aistudio.google.com/app/apikey
GEMINI_API_KEY=your_gemini_api_key_here

# RapidAPI Key (可选，备用爬虫)
# 获取地址：https://rapidapi.com/
RAPIDAPI_KEY=your_rapidapi_key_here
RAPIDAPI_HOST=real-time-amazon-data.p.rapidapi.com
```

### 第二步：一键启动

双击运行：**`客户演示-一键启动.bat`**

脚本会自动：
- ✅ 检查环境（Node.js、配置文件）
- ✅ 安装依赖（首次启动需要5-10分钟）
- ✅ 启动后端服务（端口3001）
- ✅ 启动Web报告页面（端口3002）
- ✅ 自动打开浏览器

### 第三步：开始使用

**方式1：Chrome扩展（推荐）**
1. 安装Chrome扩展（参考下方"安装Chrome扩展"）
2. 访问任意Amazon产品页面
3. 点击扩展图标
4. 点击"开始AI分析"
5. 查看分析结果或点击"查看完整报告"

**方式2：直接使用Web报告**
1. 浏览器会自动打开 http://localhost:3002
2. 点击"创建新分析"
3. 输入Amazon产品链接
4. 查看详细报告

## ⚠️ 常见问题

### Q1: 启动脚本报错"未安装Node.js"？

**A**: 请先安装Node.js
- 下载地址：https://nodejs.org/
- 推荐版本：16.x 或更高

### Q2: 启动脚本报错"未找到.env文件"？

**A**: 请按照"第一步：配置API密钥"操作
1. 复制`env.example`为`.env`
2. 填写GEMINI_API_KEY

### Q3: Chrome扩展显示"后端连接失败"？

**A**: 确保后端服务正常运行
1. 检查【后端API】窗口是否打开
2. 访问：http://localhost:3001/api/health
3. 应该看到：`{"status":"OK"}`

### Q4: Web报告页面空白？

**A**: 
1. 检查【Web报告】窗口是否打开
2. 刷新页面（F5）
3. 清除浏览器缓存后重试

### Q5: 首次启动很慢？

**A**: 正常现象
- 首次启动需要安装依赖（5-10分钟）
- 后续启动会很快（几秒钟）

### Q6: 如何停止服务？

**A**: 两种方式
1. 运行`客户演示-停止服务.bat`
2. 关闭【后端API】和【Web报告】两个窗口
