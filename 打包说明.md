## 🚀 打包步骤

### 方法1: 手动打包（推荐）

1. **清理项目**
   ```bash
   # 删除依赖
   Remove-Item -Recurse -Force node_modules, web\node_modules -ErrorAction SilentlyContinue
   
   # 删除.env（用户自己配置）
   Remove-Item .env -ErrorAction SilentlyContinue
   ```

2. **创建压缩包**
   - 选中整个`maijiaplug`文件夹
   - 右键 → "发送到" → "压缩(zipped)文件夹"
   - 重命名为：`Amazon评论分析系统-客户演示包-v1.0.zip`

3. **验证压缩包**
   - 解压到新目录
   - 运行`客户演示-一键启动.bat`
   - 确保能正常启动

### 方法2: 使用Git导出

```bash
# 导出干净的代码（不包含.git）
git archive --format=zip --output=../Amazon评论分析系统-客户演示包-v1.0.zip HEAD
```

### 方法3: 使用PowerShell脚本

```powershell
# 压缩项目（排除不需要的文件）
$source = "."
$destination = "..\Amazon评论分析系统-客户演示包-v1.0.zip"
$exclude = @("node_modules", ".git", "_archive", ".env", "*.log", "web\node_modules", "web\dist")

# 使用7-Zip或WinRAR（如果已安装）
# 或使用Windows内置的Compress-Archive
Compress-Archive -Path $source -DestinationPath $destination -Force
```
