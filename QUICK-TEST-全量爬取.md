## 🔧 故障排查（2分钟）

### ❌ 问题：仍然只爬100条

**解决方法：**

```bash
1. 确认已重启后端（Ctrl+C 然后重新运行 START-BACKEND.bat）
2. 检查日志是否显示"全量（无限制）"
3. 如果仍然显示"500条"，说明服务未重启成功
```

### ❌ 问题：日志显示"500条"

**原因：** 服务没有重启，仍在使用旧代码

**解决方法：**

```bash
# 完全关闭并重启
1. 关闭后端窗口
2. 等待3秒
3. 双击 START-BACKEND.bat 重新启动
4. 重新测试
```
