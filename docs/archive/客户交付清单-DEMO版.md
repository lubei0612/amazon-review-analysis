## 🚀 使用流程

### 首次使用

```
1. 安装依赖
   npm install
   cd web && npm install

2. 配置API密钥
   复制 .env.example 为 .env
   填入 GEMINI_API_KEY 和 OUTSCRAPER_API_KEY

3. 启动系统
   双击 "一键启动-客户版.bat"

4. 开始使用
   访问 http://localhost:3002
```

### 日常使用

```
1. 启动
   双击 "一键启动-客户版.bat"

2. 使用
   Web端: 输入ASIN → 点击分析 → 查看报告
   Chrome: 访问Amazon页面 → 使用插件

3. 停止
   双击 "停止所有服务.bat"
```

## ⚠️ 已知限制

### Outscraper API不稳定性
**现象**: 同一ASIN时好时坏  
**原因**: API本身不稳定  
**影响**: 成功率约60%  
**应对**: 等待重试 + 配置备用爬虫

### Puppeteer被拦截
**现象**: 被Amazon登录墙阻挡  
**原因**: 无登录状态  
**影响**: 成功率约30%  
**应对**: 作为最后备选

### RapidAPI状态
**现象**: 官方修复中  
**状态**: 暂不可用  
**应对**: 可选配置，不影响主要功能
