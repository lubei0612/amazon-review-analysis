```

### 3️⃣ 开始分析

```
1. 点击浏览器工具栏的扩展图标
2. 点击"🚀 开始AI分析"按钮
3. 等待约 50 秒（爬取 + AI 分析）

## ❓ 如果测试失败

### 问题 1: 仍然显示苹果切割器内容

**解决方案**：
```bash
# 1. 重启后端服务器
Ctrl+C （停止当前服务）
npm start （重新启动）

# 2. 清除浏览器缓存
# 3. 重新加载扩展
# 4. 重新分析
```

### 问题 2: 放大弹窗仍有省略号

**解决方案**：
```
1. 确认已重新加载扩展（chrome://extensions/）
2. 清除浏览器缓存：
   - F12 -> Network -> 勾选 "Disable cache"
3. 刷新 Amazon 页面（Ctrl+F5）
4. 重新分析
```

### 问题 3: 扩展无法加载

**解决方案**：
```
1. 检查后端是否运行：http://localhost:3001/api/health
2. 如果没有运行：npm start
3. 检查端口占用：netstat -ano | findstr :3001
4. 如果占用：taskkill /PID <PID> /F
```
