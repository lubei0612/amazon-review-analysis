## 🚀 快速使用指南

### 日常开发

```bash
# 启动完整系统
START.bat

# 或分步启动：
npm start         # 后端
cd web && npm start  # 前端（新终端）
```

### 客户演示

```bash
# 一键启动（推荐）
客户演示-一键启动.bat

# 停止服务
客户演示-停止服务.bat
```

### 遇到问题时

```bash
# 重启后端服务
重启服务.bat

# 或手动：
# 1. Ctrl+C 停止后端
# 2. npm start 重启
```
