# 🚀 快速开始 - Amazon 评论分析系统

> **3 分钟启动指南**

---

## 📋 前提条件

- ✅ Node.js 已安装
- ✅ Chrome 浏览器
- ✅ API Keys 已配置（`.env` 文件）

---

## 🎯 一键启动（推荐）

### 方法 1：客户演示模式

```bash
客户演示-一键启动.bat
```

这会自动：
1. ✅ 启动后端服务器（端口 3001）
2. ✅ 启动前端界面（端口 3002）
3. ✅ 打开浏览器

### 方法 2：标准启动

```bash
START.bat
```

---

## 🔧 手动启动（开发模式）

### 1️⃣ 启动后端

```bash
npm start
```

看到这个说明成功：
```
🚀 服务器运行在: http://localhost:3001
```

### 2️⃣ 启动前端（新终端）

```bash
cd web
npm start
```

浏览器会自动打开：`http://localhost:3002`

---

## 🌐 使用 Chrome 扩展

### 1. 安装扩展

1. 打开 Chrome: `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `chrome-extension` 文件夹
5. 确认扩展已启用

### 2. 使用扩展

1. 访问 Amazon 产品页
   - 例如: https://www.amazon.com/dp/B0D9JBGWCL
2. 页面会自动注入分析面板
3. 点击"🚀 开始AI分析"
4. 等待约 50 秒
5. 查看 6 个维度的分析结果

---

## ⚠️ 遇到问题？

### 重启服务

```bash
重启服务.bat
```

或手动：
```bash
# 停止当前服务（Ctrl+C）
# 重新启动
npm start
```

### 停止所有服务

```bash
客户演示-停止服务.bat
```
